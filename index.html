Projects

Most recent works

OptumRx Benefit Operation Management (BOM) Tool

OptumRx BOM Tool is an advanced AI-integrated automation engine designed in multiple microservices to process and resolve ServiceNow tickets end-to-end, reducing manual intervention and improving operational efficiency. The system fetches ServiceNow tickets and uses AI integration for intent identification via LLM-based classification to dynamically execute relevant queries and create tasks with automated validation before closing. Implemented with intelligent decision-making for multi-step execution and contextual accuracy. Developed a Java Spring Boot-based backend with a web portal for real-time monitoring of processed tickets, task status, and BOM-driven automation outcomes. Built with Java, Spring Boot, LLMs, Kafka, REST APIs, and Distributed Messaging Architecture for scalable automation.

Optum Speciality Fusion Engine(OSFE)

OSFE (Optum Specialty Fusion Engine) is a healthcare decision-support platform built for Optum Rx to enhance the Prior Authorization (PA) process. It optimizes medical and pharmacy benefit decisions by analyzing member eligibility, benefit coverage, clinical guidelines, and large-scale historical claims data. The platform recommends cost-effective and clinically appropriate alternatives for medications, treatments, and Sites of Care (SOC). By integrating data-driven insights into PA workflows, OSFE improves authorization efficiency, treatment appropriateness, and overall healthcare cost outcomes.

Real Time Agent Assist (RTAA)

RTAA is a real-time call intelligence and agent assist platform designed for live contact center interactions. During an active call, audio is streamed directly into the system, where real-time speech recognition generates live transcripts. The platform continuously analyzes conversations to detect key phrases and triggers cue cards or guidance for agents in real time. This helps agents maintain compliance, follow scripts, and respond more effectively during customer interactions. RTAA provides live insights and contextual assistance throughout the call. After the call ends, the complete interaction is automatically sent for deeper post-call analysis, quality evaluation, and reporting.

Virtual Interaction Analyst (VIA)

VIA (Virtual Interaction Analyst) is an AI-powered call analytics platform developed for MIHUP, a Kolkata-based tech startup, designed for call center and contact center environments. The platform integrates with multiple telephony partners to capture and analyze call recordings end-to-end. VIA processes calls through speaker diarization and automatic speech recognition (ASR) to generate accurate transcripts. It performs transcript, audio, and text analysis including key phrase spotting, compliance monitoring, and mood detection. The system provides tools for quality control (QC) evaluation, agent performance assessment, and compliance tracking. Detailed dashboards and reports help organizations gain actionable insights to improve service quality and customer experience.

Teams Creation Wizard

Teams Creation Wizard is a web application designed to efficiently create and manage teams based on specific business requirements. The application streamlines team formation while seamlessly integrating with multiple enterprise workflow processes. It is deployed both as a standalone web application and as a plugin within Microsoft Teams. The system leverages internal company databases and Microsoft Graph APIs to automate team creation, membership management, and collaboration setup. The backend integrates with multiple external platforms using REST APIs and Kafka, enabling scalable orchestration of team and folder management across systems.

Symphony Chat Custom Application Development

Symphony Chat–based solution consisting of multiple embedded custom applications, including BCMA WhatsApp, BCMA WeChat, and BCMA SMS. These applications use Symphony APIs to integrate external communication channels such as WhatsApp, WeChat, and SMS into the Symphony Chat platform. The system leverages the company database to manage contact and user information, enabling seamless cross-channel messaging within a unified collaboration environment. Built on a Java Spring Boot microservices architecture, the platform supports scalable, secure, and reliable communication across multiple channels.

Jira Report Automation

The Jira Automation Report system is designed to enhance and monitor project management processes by validating issue data against predefined conditions. It automatically analyzes Jira records to ensure data accuracy and compliance with organizational standards. The system generates multiple structured reports in spreadsheet format for management review and decision-making. In case of discrepancies or rule violations, it triggers automated notifications to relevant stakeholders. By automating validations, reporting, and alerts, the platform streamlines workflows and significantly reduces manual effort while improving visibility and governance.

Online Examination System

The Online Examination System is a comprehensive platform designed to conduct and manage digital assessments efficiently. It provides features for exam creation, scheduling, user management, and automated result processing. The system ensures a smooth and intuitive experience for both administrators and participants throughout the examination lifecycle. Built with a focus on scalability, reliability, and security, it supports large numbers of concurrent users and protects sensitive examination data. The platform is suitable for educational institutions and organizations seeking a dependable solution for conducting online examinations.
